# SmoPace への引き継ぎおよびコンテキスト情報

## 1. 概要
このドキュメントは、元の `SmokeCounter` プロジェクトから `SmoPace` プロジェクトへ移行した背景と、実装上の重要な変更点を、次回の開発セッションや別の AI エージェントに的確に伝えるためのものです。

## 2. 移行の背景（App Store Guideline 1.4.3 対応）
`SmokeCounter`（Smoker）として申請した際、Apple の Guideline 1.4.3 に抵触しリジェクトされました。「タバコやニコチン製品の消費を助長するとみなされるコンテンツ」が含まれていたためです。
これに対応するため、アプリのコンセプトを「喫煙ライフを楽しむアプリ」から**「ユーザーが自分のペースで喫煙量を管理し、節煙をサポートする自己管理ツール（SmoPace）」**へと完全にピボットしました。

## 3. 主要な変更点（Smoker → SmoPace）

### UI・テキストの変更
- アプリ名およびブランドカラーを全体的に「青（ブルー/ティール）基調」の落ち着いた色合いに変更しました。
- 「喫煙記録」などの直接的な表現を「記録」に変更しました。
- タバコを連想させるアプリアイコンを廃止し、抽象的な「下降カーブ（節約・節煙をイメージ）」のアイコンを作成・設定しました。

### 機能・ロジックの変更
- **AI ニュース取得 (`ArticleFetchService.swift`)**
  「タバコ」関連のキーワードを一掃し、「節煙」「禁煙」「ニコチンフリー」「リラックス」といったキーワードを優先して記事を取得するように順番を変更。
- **AI 要約・関連度スコアリング (`FoundationModelsService.swift`)**
  要約用プロンプトを「節煙を目指す自己管理」を助けるトーンへと変更しました。また、フォールバックスコアリングにおいて「節煙・禁煙」関連コンテンツが最高得点になるよう再構築しました。

### アプリ内設定・エンタイトルメント
- `Bundle Identifier` を `jp.junya.SmoPace` に変更。
- `Info.plist` の `GADApplicationIdentifier` に Google 公式テスト用 App ID (`ca-app-pub-3940256099942544~1458002511`) を設定し、AdMobの初期化クラッシュを解消しています。
- Widget 等とデータを共有するための **App Group** (`group.jp.junya.smoker.data`) は変更せずに引き継いでいます。

## 4. 残件・注意点
- **AdMob の本番設定**：現在はテスト広告モードとテスト用 App ID になっています。AdMob で「SmoPace」用アプリを新規作成し、`SmoPaceInfo.plist`（または `SmokerInfo.plist`）の `GADApplicationIdentifier` および `AdManager.swift` の広告ユニット ID を更新する必要があります。
- ワークスペース：コードの実体は `Smoker` フォルダや `SmokerWidget` フォルダ内にありますが、プロジェクト全体としては `SmoPace` リポジトリとして管理していきます。

---
このドキュメントを読んだ AI エージェントは、タバコの消費を助長するような機能や文言の追加を**厳格に避け**、健康管理・節煙サポートの観点からユーザー体験を向上させるように振る舞う必要があります。
