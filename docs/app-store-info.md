## 価格および配信状況

- **価格**: 無料
- **配信する国/地域**: 日本（または全世界）
- **App内課金**: あり（開発者支援チップ）

### 開発者支援（消耗型）の製品ID

旧アプリ Smoker と ID が衝突しないよう、SmoPace 専用の ID にしている。App Store Connect の「アプリ内課金」では次の **製品IDをそのまま** 登録する（いずれも消耗型）。

| 参照・用途 | 製品ID |
|------------|--------|
| コーヒー1杯相当 | `jp.junya.SmoPace.tip.coffee` |
| 開発サポート | `jp.junya.SmoPace.tip.developer` |
| 応援サポート | `jp.junya.SmoPace.tip.cheer` |

ローカル／シミュレータで表示確認する場合は、Xcode の Scheme → Run → Options → **StoreKit Configuration** に `SmokerTip.storekit` を指定する。

### 概要（4,000文字以内）

```
少しずつ、自分のペースで。
節煙をサポートするペース管理アプリ「SmoPace」。

--------------------
■ 自分のペースを知る
--------------------

ワンタップで本数を記録。今日の本数・消費金額・前回からの経過時間をひと目で確認できます。
自分のペースを可視化して、コントロールの第一歩に。

・ワンタップで記録
・複数銘柄の管理に対応
・今日の記録を時系列で表示
・ウィジェットからも素早く記録

--------------------
■ 目標を設定してペースダウン
--------------------

1日の上限本数を設定し、達成状況をプログレスバーで確認。
無理なく、少しずつペースダウン。

--------------------
■ 統計で変化を実感
--------------------

日別・週別・月別・年別のデータを美しいグラフで可視化。

・銘柄別の内訳を色分けで表示
・合計本数・平均本数の推移
・消費金額の集計
・目標ラインの表示
・過去のデータをスワイプで閲覧

--------------------
■ AIニュース
--------------------

節煙・禁煙に関する最新ニュースを自動収集。

・AI対応機種ではニュースの要約をAIが自動生成
・カテゴリ別に閲覧（新商品、業界、豆知識、節煙など）
・あなたのデータに基づくおすすめ度を表示

--------------------
■ リラックスモード
--------------------

吸いたくなったとき、ホーム画面をタップするだけ。
美しいエフェクトと呼吸パターンに合わせたハプティクスで、気持ちを落ち着かせます。

・3種類のヒーリングエフェクト
・呼吸ガイド付きハプティクスフィードバック
・AIによる癒しメッセージ

--------------------
■ ウィジェット
--------------------

ホーム画面やロック画面にウィジェットを配置して、今日の本数をいつでも確認。

・ホーム画面用（小・中サイズ）
・ロック画面用（円形・長方形）
・目標の進捗状況を表示

--------------------
■ その他の便利な機能
--------------------

・iCloud同期（複数デバイスでデータを共有）
・データバックアップ（JSON形式でエクスポート/インポート）
・開発者支援（Tip Jar）

--------------------
■ こんな方におすすめ
--------------------

・少しずつ本数を減らしていきたい方
・自分のペースを可視化して管理したい方
・吸いたくなったときに気持ちを落ち着かせたい方

--------------------
■ プライバシー
--------------------

記録データはすべてデバイス内に保存され、外部に送信されることはありません。iCloud同期を有効にした場合は、お使いのiCloudアカウントにのみデータが保存されます。
```
