## バージョン 1.0 情報

### アプリ説明（日本語）

```
喫煙本数を簡単に記録できるアプリです。

■ 主な機能
・ワンタップで喫煙を記録
・日別/週別/月別の統計表示
・ウィジェット対応で素早くアクセス
・目標設定と達成状況の確認

■ こんな方におすすめ
・禁煙を目指している方
・喫煙量を把握したい方
・健康管理を意識している方

シンプルな操作で、毎日の喫煙習慣を可視化できます。
データはすべてデバイス内に保存され、外部に送信されることはありません。
```
