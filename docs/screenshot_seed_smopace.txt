スクリーンショット用サンプルデータ（screenshot_seed_smopace.json）
================================================================

■ 内容
  - 銘柄: Teria / マイセン / ラッキー（各20本入り・価格はサンプル）
  - 1日の目標本数: 10本
  - 記録期間: 2026年3月1日 〜 4月18日（全日にばらつきある本数）
  - 統計の「週」「月」で棒グラフと銘柄内訳が分かりやすいよう日別に配分

■ 取り込み手順
  1. 可能なら「データ管理」で iCloud同期をオフにしてから試す（シミュレータでストア競合が起きにくい）
  2. 設定タブ → 「データのバックアップと復元」→「データをインポート」
  3. この JSON（screenshot_seed_smopace.json）を選ぶ
     （確認ダイアログで既存データは上書きされます）
  4. 同じエラーが出る場合：ホーム画面から SmoPace ウィジェットを外す → アプリを削除して再インストール → 再度インポート

■ 撮影のコツ（日付について）
  統計の「今週」「今月」は端末のカレンダー基準です。
  データは 2026年3〜4月向けなので、シミュレータまたは端末の日付を
  「2026年4月13日」前後に設定すると、週次・月次（4月）の見え方が意図に近くなります。

■ 補足
  インポート時に銘柄・記録の UUID を保持するようアプリ側を修正済みです。
  このファイルを編集する場合は、records 内の brandId と brands 内の id の対応を
  崩さないようにしてください。
