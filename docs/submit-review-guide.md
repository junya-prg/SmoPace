# App Store 審査提出ガイド

---

## 提出前チェックリスト

審査に提出する前に、以下の項目がすべて完了していることを確認してください。

### 必須項目

- [ ] アプリ名が入力されている
- [ ] サブタイトルが入力されている（任意だが推奨）
- [ ] アプリ説明が入力されている
- [ ] キーワードが入力されている
- [ ] サポートURLが入力されている
- [ ] プライバシーポリシーURLが入力されている
- [ ] スクリーンショットがアップロードされている（各デバイスサイズ）
- [ ] アプリアイコンが設定されている
- [ ] カテゴリが選択されている
- [ ] 年齢制限が設定されている
- [ ] 価格が設定されている
- [ ] ビルドが選択されている
- [ ] Appのプライバシーが設定されている

---

## 審査提出手順

### Step 1: App Store Connect にアクセス

1. [App Store Connect](https://appstoreconnect.apple.com) にログイン
2. マイApp → Smoker を選択

### Step 2: すべての情報を確認

「App Store」タブで、バージョン 1.0 のすべての情報が入力されていることを確認します。

### Step 3: 「審査へ提出」をクリック

ページ右上の **「審査へ提出」** ボタンをクリックします。

### Step 4: 輸出コンプライアンス

**質問**: 「このAppは暗号化を使用していますか？」

**回答**: **いいえ**

理由: 標準的なHTTPS通信のみを使用しており、独自の暗号化アルゴリズムは使用していません。

### Step 5: 広告識別子（IDFA）

**質問**: 「このAppは広告識別子（IDFA）を使用しますか？」

**回答**: **はい**

**使用目的を選択**:
- [x] このApp内で広告を配信する

理由: Google AdMobを使用して広告を表示しているため。

### Step 6: コンテンツ配信権

**質問**: 「このAppは第三者のコンテンツを配信しますか？」

**回答**: **いいえ**

### Step 7: 提出完了

すべての質問に回答後、**「提出」** をクリックします。

---

## 審査ステータス

提出後、以下のステータスで進行状況を確認できます：

| ステータス | 説明 |
|------------|------|
| 審査待ち | 審査キューに入っています |
| 審査中 | Appleが審査を行っています |
| 承認済み | 審査に合格しました |
| リジェクト | 審査に不合格でした（理由が通知されます） |
| デベロッパによるリリース待ち | 手動リリース設定の場合 |

---

## 審査期間

- **通常**: 24〜48時間
- **初回アプリ**: 最大1週間程度
- **繁忙期（年末年始など）**: さらに長くなる場合あり

---

## 審査でよくあるリジェクト理由と対策

### 1. メタデータの問題

**理由**: スクリーンショットが実際のアプリと異なる

**対策**: 実機またはシミュレータで撮影した実際の画面を使用

### 2. クラッシュ

**理由**: 審査中にアプリがクラッシュした

**対策**: 十分なテストを行い、クラッシュを修正してから再提出

### 3. プライバシー関連

**理由**: プライバシーポリシーが不十分、またはATT実装が不適切

**対策**: 
- プライバシーポリシーに収集データを明記
- ATTダイアログを適切なタイミングで表示

### 4. 機能不足

**理由**: アプリの機能が少なすぎる

**対策**: 基本的な機能が動作することを確認

### 5. 広告関連

**理由**: 広告が多すぎる、または不適切な位置に表示

**対策**: ユーザー体験を損なわない広告配置

---

## リジェクトされた場合

1. **理由を確認**: App Store Connect の「解決センター」でリジェクト理由を確認
2. **修正**: 指摘された問題を修正
3. **再提出**: 新しいビルドをアップロードして再提出

---

## 承認後のリリース

### 自動リリース

「バージョンのリリース」で「審査後、自動的にリリース」を選択している場合、承認後すぐにApp Storeに公開されます。

### 手動リリース

「手動でリリース」を選択している場合、承認後に「このバージョンをリリース」ボタンをクリックする必要があります。

---

## 公開後

アプリが公開されると、以下のURLでアクセスできます：

```
https://apps.apple.com/jp/app/smoker/id[APP_ID]
```

※ APP_ID は App Store Connect で確認できます。
