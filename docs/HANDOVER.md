# SmoPace プロジェクト引き継ぎ・コンテキストドキュメント

## 1. プロジェクトの概要
本プロジェクト **SmoPace（スモペース）** は、ユーザーが自分のペースで喫煙量を管理し、節煙（タバコの消費量削減）をサポートするための iOS アプリケーションです。

## 2. 移行とピボットの背景
元々は「喫煙ライフを楽しむアプリ」である `SmokeCounter`（Smoker）として開発されましたが、App Store Guideline 1.4.3（タバコやニコチン製品の消費を助長するコンテンツの禁止）に抵触し、リジェクトされました。
このリジェクトに対応するため、アプリのコンセプトを完全に「ユーザーのペースに合わせた自己記録・健康管理（節煙サポート）」へとピボットし、名称も `SmoPace` に変更しました。

## 3. アーキテクチャとファイル構造における注意点
- **ワークスペース/プロジェクト名:** ルートプロジェクトは `SmoPace.xcodeproj` です。
- **内部フォルダ・スキーム名について:** ピボット前の名残として、プロジェクト内には依然として `Smoker/` や `SmokerWidget/` というフォルダ、また `Smoker` という名前のスキームや設定ファイル（`SmokerTip.storekit`, `Smoker.entitlements`, `SmokerWidgetInfo.plist`等）が存在しています。
  > [!WARNING]
  > **これらを安易にリネーム・削除しないでください。**
  > Xcode の `project.pbxproj` と深く紐付いているため、ファイルシステム上でフォルダ名だけを変更するとビルドが完全に壊れる原因となります。これらは歴史的経緯としてそのままにしておくのが安全です。（※完全に不要となった `SmokerInfo.plist` のみは削除済みです）
- **App Group:** ウィジェットとデータを共有するための App Group (`group.jp.junya.smoker.data`) も、以前の設定まま変更せずに引き継いでいます。

## 4. 実装における厳格なルール
- **UI & テキスト:** アプリ全体は青（ブルー/ティール）を基調とした落ち着いた色合いです。また、「喫煙記録」などの直接的な用語は単に「記録」へと表現を和らげています。
- **タバコ消費の助長禁止:**
  新機能の追加や AI プロンプトの修正、テキストの変更を行う際は、**タバコの消費を助長する表現を厳格に避けてください。**
  `ArticleFetchService.swift` や `FoundationModelsService.swift` では、AI関連のキーワードが「節煙」「禁煙」「ニコチンフリー」のトピックに偏る（これらが最高得点になる）ようハードコードされています。

## 5. 今後の残件タスク
- **AdMob の本番設定:** 
  現在は、起動時のクラッシュを防ぐために Google 公式のテスト用 App ID (`ca-app-pub-3940256099942544~1458002511`) を `SmoPaceInfo.plist` に設定しています。本番リリース前には、AdMob 上で「SmoPace」用の新しいアプリを作成し、`SmoPaceInfo.plist` の `GADApplicationIdentifier` および `AdManager.swift` の広告ユニット ID を本番用のものに確実に更新する必要があります。
